    Help File

    Purpose

    The purpose of Mountain Top Mining Explorer is to provide a map that allows the user to select a possible mining location in southeastern Idaho and a proposed elevation for a mining cut. The Mining Explorer App then computes the volume of material to be removed and creates an updated DEM showing the cut. Large efforts were made to make this app user friendly.

    This app was created to fulfill a project assignment in the GIS Programming for Engineers Class taught by Dr. Ames at Brigham Young University.

    Required Software

    Tethys

    Tethys is a framework to build web apps. It runs on Linux and requires the installation of many dependencies before it can run. For more information about how to install Tethys go to: http://docs.tethysplatform.org/en/latest/installation.html.

    Virtual Box

    A workaround to be able to install Tethys without a Linux machine is to create a virtual machine that runs Linux. The following link has information about how to download and install a virtual machine using oracleâ€™s VirtualBox: https://www.virtualbox.org/wiki/Downloads.

    Compatible Web Browsers

    This app uses basic HTML, CSS, and JavaScript code. Mining Explorer had been tested and found compatible in Firefox and Google Chrome.

    ArcMap 10.3

    ArcMap was used to create the initial model. Once the model was completed it was uploaded to the BYU GIS Server so that it could be pulled into the Tethys app.

    Geoserver

    An ArcMap tool was first made that ran the functions of the tool. This tool was then uploaded to http://geoserver.byu.edu/arcgis/rest/services/MineExplorer. Please view the website for more information about how the tool runs.

    Coding Interface

    Pycharm was the coding interface used to write the code. However other coding interfaces are available and can be used.

    The code is accesible for viewing in GitHub at https://github.com/byuceen514/MineExplorer.git.

    Required Input Data

    All of the data needed to create Mining Explorer can be found at gis.utah.gov. The data used was the Idaho DEM.

    Configuring the Data

    The DEM file has been previously inserted into the Mining Explorer App. All that the user needs to import is the location and the desired elevation of the cut.

    Restrictions

    The geospatial data seen in this app was downloaded from the AGRC website. This website is not intended to be a comprehensive or even accurate description of mountain top mining. Rather, this is a limited collection of selected events with incomplete and possibly incorrect information.

    Using the App

    Menu

    The menu is located at the left-hand side of the page. And contains the links: Home, Map, Help File, and Design Documentation. Below is a description of each page.

    Home

    The Home page briefly discusses the purpose of the site, what it contains, and how to navigate through the website.

    Map

    The map allows the user to draw a polygon around a desired mining location. It then prompts the user to input a cut elevation. Following these steps it outputs an updated DEM showing the cut.
